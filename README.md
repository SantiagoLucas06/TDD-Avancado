# TDD-Avancado

Classe Fibonacci: Define uma classe para calcular o n-ésimo termo da sequência de Fibonacci.
Método fibonacciRecursivo(int n): Calcula o n-ésimo termo da sequência de Fibonacci de forma recursiva. Se n for menor ou igual a 1, retorna n. Caso contrário, calcula recursivamente somando os dois termos anteriores.
Comentários Javadoc: Os comentários Javadoc foram utilizados para documentar a classe e o método, explicando seu propósito, parâmetros e retorno.
Declaração de parâmetro e retorno: Cada parâmetro e o retorno do método são documentados usando anotações Javadoc, explicando sua finalidade.
Condição de base: Uma condição de base é verificada para os casos em que n é 0 ou 1, retornando n diretamente sem recursão.
Recursão: Para valores de n maiores que 1, a recursão é utilizada para calcular o valor do n-ésimo termo somando os dois termos anteriores.
Esta documentação ajuda a entender rapidamente o propósito e o funcionamento do método fibonacciRecursivo, facilitando seu uso e manutenção.
